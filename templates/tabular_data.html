Tabular Data

Upload, process, query, and visualize tabular data files. You can manage your data and create insightful visualizations with ease.

Upload File

Process Data

Select a file and the type of statistics you want to generate. Basic statistics include mean, median, etc., while advanced statistics include outliers, skewness, and more.

Query Data

Run SQL-like queries on your data to extract specific information. Make sure to provide a valid SQL SELECT statement.

Manage Data

You can create, read, update, and delete entries in your data file. Input the data in JSON format.

Visualize Data

Generate different types of plots for your data. You can create histograms, bar charts, and box plots.
